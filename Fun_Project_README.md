# Number Guessing Game

## Description
This project is a simple Python-based number guessing game where the system generates a random number between 1 and 100. The user has to guess the number within a limited number of attempts. The program provides hints whether the guess is too high or too low.

## Dataset
- Source: Not applicable (Random number generation)
- Description of data: The number is randomly generated using Python's random module.

## Steps Performed
1. Generated a random number using random.randint()
2. Took user input using input()
3. Used conditional statements (if-else) to compare guesses
4. Implemented a loop to allow multiple attempts
5. Displayed hints (Too high / Too low)

## Results
- The game successfully allows users to guess the number
- Provides interactive feedback for each attempt
- Displays win or loss message

## Tools Used
- Python
- random module

## Conclusion
This project helped in understanding basic Python concepts like loops, conditional statements, and user input handling.
