# Data Science Lab Work

## Description

This repository contains all the lab exercises and projects completed as part of the Data Science course. The labs cover basic to intermediate Python and data science concepts such as NumPy, Pandas, regression, and classification techniques.

---

## Lab Exercises

### 1. Introduction to Python

* Basics of Python programming
* Variables, data types, and input/output
* Simple programs

### 2. Control Statements

* If-else conditions
* Loops (for, while)
* Decision making programs

### 3. NumPy

* Array creation and operations
* Mathematical functions using NumPy
* Basic numerical computations

### 4. Pandas

* DataFrames and Series
* Data manipulation and analysis
* Handling missing values

### 5. Regression Analysis

* Understanding regression concepts
* Simple linear regression implementation
* Model evaluation

### 6. KNN Classification

* K-Nearest Neighbors algorithm
* Classification of data
* Accuracy evaluation

---

## Projects

### 1. Fun Project – Number Guessing Game

* A simple Python game where the user guesses a randomly generated number
* Provides hints like "Too high" or "Too low"
* Limited number of attempts

### 2. Intermediate Project – Student Marks Analyzer

* Analyzes student marks
* Calculates total and average
* Finds highest and lowest marks
