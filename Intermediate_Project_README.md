# Student Marks Analyzer

## Description
This project analyzes student marks using Python. It calculates total marks, average marks, and identifies the highest and lowest scores. It helps in understanding basic data analysis concepts.

## Dataset
- Source: Self-created dataset
- Description of data: A list of student marks entered manually in the program

## Steps Performed
1. Created a list of student marks
2. Calculated total marks using sum()
3. Computed average marks
4. Found maximum and minimum marks
5. Displayed all results clearly

## Results
- Successfully calculated total and average marks
- Identified highest and lowest scoring students
- Displayed results in a clear format

## Tools Used
- Python

## Conclusion
This project helped in understanding list operations, basic data analysis, and Python functions like sum(), max(), and min().
